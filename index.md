## Going Zero for Mac

Going Zero for Mac is a Mac application which enable you to do some DJ tricks like scratch, loop ,, on any streaming service like Apple Music/Spotify/Youtube

trick list:
- Scratch
- Loop
- Reverse
- Cue
- Refrain
- DJ Filter
- Flanger
- Random
